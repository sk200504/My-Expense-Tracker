💰 Student Expense Tracker

 📌 Project Description
The Student Expense Tracker is a web-based application developed to help students manage and track their daily expenses efficiently. The system allows users to input expense details, categorize them, and visualize their spending patterns.
This project provides a simple and effective solution for monitoring financial activities and improving budgeting habits.

🎯 Problem Statement
Students often find it difficult to keep track of their daily expenses, which can lead to poor financial planning. The lack of a structured system results in unorganized spending records.
This project addresses this issue by offering a centralized platform where users can record, view, and analyze their expenses in an organized manner.

 🚀 Features
* Add expenses with date, category, and amount
* Store expense data in a structured format
* Display all expenses in tabular form
* Automatically calculate total expenses
* Visualize category-wise spending using charts

 🛠️ Tech Stack
* Python
* Streamlit
* Pandas

 📂 Project Structure
expense-tracker/

⚙️ Environment Setup
Ensure the following software is installed on your system:
* Python (version 3.x)
* pip (Python package manager)

▶️ Step-by-Step Execution Instructions
 Step 1: Clone the Repository
```
git clone https://github.com/your-username/expense-tracker
```
 Step 2: Navigate to Project Directory
```
cd expense-tracker
```
 Step 3: Install Required Dependencies
```
python -m pip install -r requirements.txt
```
 Step 4: Run the Application
```
python -m streamlit run app.py
```
 Step 5: Access the Application

After running the above command, the application will start and open in your default browser.

If it does not open automatically, visit:
```
http://localhost:8501
```

▶️ Working of the Project

1. The user selects a date and enters expense details
2. The user selects a category (Food, Travel, Shopping, etc.)
3. The expense amount is entered
4. Data is stored in a CSV file (`expenses.csv`)
5. The system displays:

   * All expenses in tabular format
   * Total expenditure
   * Category-wise bar chart

📊 Output
* Displays recorded expenses in a structured table
* Calculates and shows total expenses
* Provides graphical representation of spending patterns
