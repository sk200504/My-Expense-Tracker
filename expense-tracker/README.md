# Student Expense Tracker

## 📌 Description
This project helps students track their daily expenses in a simple way.

## 🚀 Features
- Add expenses
- View all expenses
- Total expense calculation
- Category-wise chart

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/your-username/expense-tracker

2. Go to folder:
cd expense-tracker

3. Install dependencies:
pip install -r requirements.txt

4. Run the app:
streamlit run app.py

## 📷 Output
Displays expense table and charts.
